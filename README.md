# INTO GROUP

Небольшое backend-приложение на Flask для управления задачами, ботами и общения в реальном времени.

Изначально делалось как внутренняя система, но архитектуру старался держать достаточно гибкой, чтобы можно было нормально расширять.

---

## Что есть сейчас

* регистрация и авторизация пользователей
* работа с задачами
* управление ботами
* простой чат в реальном времени (через Socket.IO)
* работа с клиентами

---

## Стек

* Python + Flask
* Flask-SocketIO
* SQLite (пока что)
* зависимости через pip / poetry

---

## Как устроен проект

Проект разбит на blueprints, чтобы не превращать всё в один большой файл.

```
.
├── app.py
├── config.py
├── app.db
│
├── routes/
│   ├── auth.py
│   ├── tasks.py
│   ├── bots.py
│   ├── chat.py
│   └── clients.py
│
├── socket_handlers/
│   └── chat_handlers.py
│
├── models/
│   └── database.py
│
└── decorator.py
```

Отдельно вынесены обработчики сокетов — так удобнее не мешать HTTP и realtime-логику.

## Запуск

```bash
git clone <repo-url>
cd project
```

```bash
python -m venv venv
source venv/bin/activate  # linux/mac
venv\Scripts\activate     # windows
```

```bash
pip install -r requirements.txt
```

Создать `.env` (если нужно):

```
SECRET_KEY=123
```

И запустить:

```bash
python app.py
```

По умолчанию всё поднимается на
http://localhost:5000

---

## Про чат

Используется Socket.IO, чтобы можно было делать нормальный realtime без костылей.

Обработчики лежат тут:

```
socket_handlers/chat_handlers.py
```

## База данных

Сейчас используется SQLite — просто и без лишнего.

Инициализация происходит при запуске, ничего отдельно поднимать не нужно.
